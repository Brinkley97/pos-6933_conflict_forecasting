## Installation
> Use the [uv package manager](https://docs.astral.sh/uv/) package manager.

1. OPTIONAL: Create a project with `uv init .` that'll default to name of directory.
   - If you already see a `.toml` file, you should be able to skip.
2. Create virtual environment with `uv venv`.
3. Activate virtual environment with `source .venv/bin/activate`.
4. Install requirements with `uv pip install -r pyproject.toml`.
5. Install `uv pip install ipykernel` so you can run the jupyter notebooks.
